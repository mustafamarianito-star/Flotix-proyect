Descargar respaldo

Descarga un archivo con todos los datos de la app (combustible, vehículos, choferes y rendiciones) para guardarlo en un lugar seguro. El proyecto gratuito de Supabase no tiene backups automáticos, así que conviene bajar uno de estos periódicamente. Nota: este archivo incluye las facturas adjuntas (guardadas como datos en la rendición).
